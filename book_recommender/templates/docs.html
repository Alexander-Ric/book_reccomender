Documentación de la API

POST /api/recommend

Devuelve recomendaciones de libros en formato JSON.

Body (JSON) de ejemplo

{
  "favorite_genre": "Fantasia",
  "min_rating": 4.0,
  "limit": 3
}

Respuesta (ejemplo)

{
  "recommendations": [
    {
      "id": 6,
      "title": "El Señor de los Anillos",
      "author": "J. R. R. Tolkien",
      "genre": "Fantasía",
      "description": "...",
      "rating": 4.9
    }
  ]
}

Endpoints disponibles

  • GET / – Formulario HTML para pedir recomendaciones.
  • POST /recommendations – Devuelve página HTML con libros recomendados.
  • POST /api/recommend – Endpoint JSON para integraciones.
  • GET /health – Comprobación rápida del estado del servicio.
  • GET /docs – Esta documentación.
